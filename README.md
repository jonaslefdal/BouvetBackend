# Bouvet APP 

Installasjons Guide
1. * gh repo clone https://github.com/jonaslefdal/BouvetBackend
    * <b>git checkout</b> -b [my-branch] 
    * Gjør jobb
    * git commit -m "My message"
    * git push (sett upstream hvis kommandoen feiler)

Start Backend Server

2. * cd BouvetBackend
    * docker compose up --build

Stop Backend Server

3. * cd BouvetBackend
    * docker compose down
